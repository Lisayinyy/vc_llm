# vc_llm — 中国大模型公司投资方中心图谱

投资人视角的"中国 12 家顶级大模型公司"知识图谱：从投资机构出发，看它投了哪些大模型公司，并穿透到背后的**关键人物（GP/操盘人）**。

## 🎯 这是什么

一个交互式 ECharts 力导向图，覆盖：

- **12 家中国顶级大模型公司**：智谱、月之暗面、百川智能、MiniMax、零一万物、阶跃星辰、生数科技、深度求索、面壁智能、衔远科技、无问芯穹、瑞莱智慧
- **150+ 投资机构**：从启明创投、红杉中国、IDG 资本，到阿里战投、蚂蚁战投、北京 AI 国资
- **6 个关键人物**：沈南鹏（红杉）、邝子平（启明）、熊晓鸽（IDG）、纪纲（蚂蚁）、蔡崇信+吴泳铭（阿里）、北京 AI 基金 GP

**3 类节点** × **4 种关系**：

| 节点 | 颜色/形状 | 含义 |
|---|---|---|
| 大模型公司 | 紫色圆 | 12 家 |
| 投资机构 | 蓝色方块（跨公司金主=金色方块） | 150+ |
| 投资个人 | 红色菱形 | 关键人物 / 自然人股东 |

| 关系 | 颜色 |
|---|---|
| 持股（工商登记） | 黄色实线 |
| 投资（公开新闻） | 橙色实线 |
| 关键人物 ↔ 机构 | 红色虚线 |
| 公司 ↔ 公司（同被投资） | 蓝色点线 |

## 🚀 怎么用

### 在线看

直接打开 `index.html` 即可，所有数据已嵌入。

### 本地重新生成

```bash
# 1. 准备 v4 数据 (集成 v3 工商数据 + web 公开融资 + 6 个关键人物)
python3 build_data_v4.py

# 2. 生成 HTML 图谱
python3 build_html.py
# → 生成 llm_graph_v6.html

# 3. 用浏览器打开
open llm_graph_v6.html
```

## 🎮 玩法

1. **默认进入**：以「启明创投」为中心（投了 4 家 12 强最多），看它投了哪些公司
2. **看老大**：点顶部「关键人物排行」——阿里系投了 5 家、邝子平（启明）投了 4 家
3. **穿透到机构背后**：点红色菱形节点（关键人物）→ 看 ta 投了哪些 + 所在机构
4. **看公司关系**：两个公司之间的蓝色点线 = 被同一投资方投了

**数据纪律**：
- 严格标注来源：🟢 工商数据 vs 🟠 公开新闻
- 不披露的写"未披露"，不补全
- 不含期权/ESOP/间接持股

## 数据准确性审计

原始 `llm_data_v4.json` 中的 `source: web` 只表示通过公开网页收集，不能视为已核实证据。审计发现 127 条公开关系均缺少逐条来源 URL、发布日期和证据摘录，并存在人物归因过度、关系类型混用、日期冲突与机构别名重复。

用于严肃分析时请运行：

```bash
python3 audit_data.py
python3 build_html.py
```

图谱生成器会优先读取 `llm_data_audited.json`。严格数据集目前只纳入用户指定可信的企查查记录，以及具备逐条一手证据的公开关系；其余记录保留在 `unverified_relationships`，但不参与关系图和排名。完整问题与准入规则见 [`DATA_AUDIT.md`](DATA_AUDIT.md)。
